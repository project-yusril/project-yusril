## Hello, I am Yusril

I build thoughtful mobile experiences with **Flutter** and turn product ideas into complete web solutions when the project calls for it. My work sits at the intersection of software engineering, practical UI design, and education.

- Mobile-first developer focused on Flutter and Dart
- Full-stack web developer working across modern JavaScript and PHP ecosystems
- Lecturer in the **Diploma III Informatics Engineering Program** at **Politeknik Negeri Pontianak**
- Based in Pontianak, West Kalimantan, Indonesia
- Interested in mobile engineering, applied computing, and building technology that solves real problems
